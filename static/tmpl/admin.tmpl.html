White list

Here you can place a list of allowed sites. You can use any set of regular expression to customize your list. Here is a few examples: 
# explicitly allow www.wikipedia.org
www.wikipedia.org
# use regex pattern to allow all google-related sites
.*google.com
# allow org domain
org$

Black list

In this textbox you can list sites which will always be rejected by {{.package}}

Time-based rules

this is a set of time-based rules for allowed sites. Please use comma-separated list of values, e.g. 
www.facebook.com,9,15
www.myspace.com,12,15

the first entry is web site address, the last two are starting and ending time, respectively, in 24 hour clock notations.
